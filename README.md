## 🚀 Instalando <Blog>

Para instalar o <Blog>, siga estas etapas:


Clone este repositório
```
git clone <nome do repositorio>
```

Instale as dependencias
```
npm install 
```

## ☕ Usando <Blog>

Para usar <Blog>, siga estas etapas:

```
npm run dev
```
